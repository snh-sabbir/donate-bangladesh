What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a document, typically an HTML or XML document. It allows developers to interact with the content and structure of a web page in a structured way.

How do you select an element from the DOM?

You can select elements from the DOM using various methods. Such as document.getElementById(), document.getElementsByClassName(), document.getElementsByTagName(), document.querySelector(), document.querySelectorAll().

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript for handling events at a higher level in the DOM rather than attaching individual event listeners to multiple elements. Instead, you attach a single event listener to a parent element, which captures events that bubble up from its child elements

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles by using getAttribute(), setAttribute(), removeAttribute() etc.
